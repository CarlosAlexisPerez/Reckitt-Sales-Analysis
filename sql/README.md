# SQL Analysis

This folder contains SQL queries developed for data exploration and business analysis using relational databases.

The project includes practical exercises focused on:
- Data querying
- Filtering and sorting
- Aggregations and grouping
- Joins and relationships
- Subqueries
- Common Table Expressions (CTEs)
- Business-oriented analysis

## Topics Covered
- Customer analysis
- Movie rental analysis
- Sales analysis
- Product and actor analysis
- Store performance metrics

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- JOIN
- LEFT JOIN
- Subqueries
- Aggregate Functions
- CTEs (WITH)
